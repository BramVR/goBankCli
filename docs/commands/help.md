# gobankcli --help

`gobankcli --help` prints the top-level command list, global flags, and a hint
to run command-specific help.

Use command-specific help for a focused usage line and flags:

```bash
gobankcli <command> --help
```

Help output is generated by Kong from the same command structs used by
`docs-command-reference`.
